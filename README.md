Overview
====
Name:StrucOGPviewer

version:1.0.0

release date: 2/18/2026

developed by Matlab R2023a

StrucOGPviewer is a software used to visualize the identification results of csv files generated by StrucOGP.

StrucOGPviewer is available freely for academic research, non-commercial or educational purposes under academic license.

Installation guide
====
Double click StrucOGPviewer.exe to start installing StrucOGPviewer.
StrucOGPviewer is developed by MATLAB and relies on Runtime. When the up-to-date Runtime is needed, a download link will show up as you install StrucOGPviewer.

How to acquire StrucOGPviewer license to activate your StrucOGPviewer software:
====
StrucOGPviewer does not require license verification, but please obtain a StrucOGP license(https://github.com/Sun-GlycoLab/StrucOGP) before using StrucOGPviewer.
